## When To Open An Issue

Open a GitHub issue for:

- reproducible bugs
- documentation gaps
- feature requests
- questions that reveal a missing or unclear project behavior

## What Helps Maintainers Most

Include:

- your environment
- the command or request you ran
- relevant request payloads or response bodies
- steps to reproduce
- expected behavior versus actual behavior

Security concerns should follow [SECURITY.md](/Users/ghassan/my-projects/skill-ledger/SECURITY.md), not public issues.
