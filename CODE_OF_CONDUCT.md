# Code Of Conduct

## Our Standard

We want SkillLedger to be a respectful, constructive project for contributors of different backgrounds and experience levels.

Examples of expected behavior:

- being respectful in disagreement
- assuming good intent while still giving direct technical feedback
- welcoming questions and documenting decisions clearly
- prioritizing the health of the project and the people working on it

Examples of unacceptable behavior:

- harassment, discrimination, or intimidation
- personal attacks
- deliberately disruptive behavior
- publishing private information without permission

## Scope

This code of conduct applies in repository discussions, issues, pull requests, and other project spaces managed by the maintainers.

## Reporting

Please report unacceptable behavior through the contact path in [SECURITY.md](/Users/ghassan/my-projects/skill-ledger/SECURITY.md) or by reaching out privately to the maintainers if the issue is conduct-related rather than security-related.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, or other contributions that do not align with this code of conduct.
