# SkillLedger — LinkedIn Survey Questions (Final)

**Core thesis:** Agents buy verified skills from authors who stake bonds for accountability. Authors invest time in quality because they earn. Trust comes from stake + deterministic code + verification.

---

### Q1 — Buyer side: Would you pay for a skill with author accountability?

> Another agent offers a "run SQL" or "scrape this URL" skill. The author stakes crypto — if the result is wrong, their bond gets slashed and you're refunded. Would your agent use this?
>
> 🅰️ Yes — I'd pay premium for verified skills with accountability
> 🅱️ Only for critical tasks — free/unstaked is fine for simple stuff
> 🅲 No — I run everything in-house
> 🅳 The ecosystem is too early — ask me in a year

---

### Q2 — Author side: Would you build and stake skills to earn?

> You write a quality agent skill. To sell it, you stake a bond (returned if execution is correct). Buyers pay per call. Would you invest time building skills for this?
>
> 🅰️ Yes — I'd invest hours if the earning potential is there
> 🅱️ Only if I already have the skill built (zero extra effort)
> 🅲 No — staking my own capital is too risky
> 🅳 I'd rather give it away for free to build reputation first

---

### Q3 — What makes a skill trustworthy enough to buy?

> You're choosing which agent skill to hire. Rank what matters most:
>
> 🅰️ Author has staked a large bond (financial accountability)
> 🅱️ The code is open-source / commit-hashed (I can see exactly what runs)
> 🅲 Author has high reputation and 500+ successful runs
> 🅳 Cryptographic proof that the code ran correctly (TEE/ZK)
> 🅴 The price is low enough that I don't care if it fails

---

### Q4 — What's the biggest blocker?

> What stops you from paying another agent for a skill today?
>
> 🅰️ No standard way for agents to pay each other
> 🅱️ No way to verify the skill ran correctly
> 🅲 No way to recover losses if the skill returns garbage
> 🅳 No directory to find trustworthy skills
> 🅴 Nothing — we already do this with API keys and trust

---

### Q5 — Which skills would you pay for?

> Which of these would you actually pay another agent to execute?
>
> 🅰️ SQL query / data retrieval
> 🅱️ Code execution / sandboxed compute
> 🅲 Web scraping / data extraction
> 🅳 Specialized analysis (compliance, legal, financial)
> 🅴 None — I'd rather build or run everything myself
