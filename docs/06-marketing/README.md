# Marketing & Outreach

This directory contains LinkedIn content, survey questions, and articles designed to validate the SkillLedger thesis with real agent developers.

---

## Documents

### LinkedIn Poll Series (5 Days)
- **File:** [linkedin-posts.md](./linkedin-posts.md)
- **Goal:** Validate core assumptions (A1-A4) through LinkedIn polls
- **Structure:** One post per day, Mon-Fri, each with an embedded poll
  - Day 1 (Mon): **The Buyer Question** — "Would you pay for verified skills?"
  - Day 2 (Tue): **The Author Question** — "Would you stake skills to earn?"
  - Day 3 (Wed): **The Trust Question** — "What makes a skill trustworthy?"
  - Day 4 (Thu): **The Blocker Question** — "What stops you from paying?"
  - Day 5 (Fri): **The Market Question** — "Which skills would you pay for?"

## Validation Protocol

1. **Post the LinkedIn series** (1 post/day for 5 days)
2. **Collect poll data** — raw percentages for each question
3. **Follow up in comments** — use the optional comment prompts
4. **Analyze results** — do they confirm or refute assumptions A1-A4?
5. **Interview the most engaged respondents** — 5 warm conversations
6. **Decision:** Go / Pivot / Kill based on signal
