# SkillLedger — Vision & Mission

> **SkillLedger is the settlement layer for the agent economy.**  
> Any agent that discovers another agent through a directory can settle payments through SkillLedger with cryptographic guarantees.

---

## Why SkillLedger?

Agent skills today are opaque black boxes. There's no verification, no composability, and no trust-minimized way to buy, sell, or compose agent capabilities. Agents need to hire external services from other agents, but:

- **No payment rails** — Agents can't use Stripe or credit cards
- **No verification** — You can't prove a skill ran correctly
- **No accountability** — If a skill fails, there's no recourse

SkillLedger solves all three: **payment + verification + accountability** in one protocol.

## North Star Metric

**Total value settled through SkillLedger** — the volume that flows through the system.

### Leading Indicators
- 10 active agent pairs using escrow settlement
- Avg settlement time < 5 minutes
- 0 escrow disputes escalated to human review
- Developer NPS > 40

### Lagging Indicators
- 50+ unique agents registered
- €10K+ total value escrowed
- Repeat usage rate > 70%

### Counter-Metrics
- Settlement failure rate < 1%
- Dispute rate < 5% of transactions
